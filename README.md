## 🎮 O que é esse projeto?
Esse é o repositório do **Robozinho**, nosso robô de batalha desenvolvido em Java para competir na arena do jogo Robocode. A lógica principal dele é se adaptar: ele reage e muda de estratégia dependendo de quantos inimigos ainda estão vivos na partida.

Porém, o **grande foco** deste trabalho não foi aprender a programar Orientado a Objetos do zero, mas sim usar esse projeto prático para **mergulhar de cabeça no versionamento de código**. Usamos o Robozinho como "cobaia" para aprender na prática como o Git e o GitHub funcionam no fluxo de uma equipe de desenvolvimento.

## 🎯 Nossos Objetivos
* Criar um robô funcional e aplicar estratégias de batalha.
* **Dominar o versionamento:** Entender o fluxo real de trabalho com Git e GitHub.
* Perder o medo de criar *branches*, fazer *commits* sem quebrar o projeto e realizar *pull requests* de forma colaborativa.

## 🛠️ Como trabalhamos
Nós dois desenvolvemos e testamos partes separadas do código. Para juntar tudo sem dor de cabeça, o Git foi essencial. Durante as aulas presenciais, discutíamos o que estava funcionando, o que precisava sair e fazíamos os *merges* do código para a versão principal.

**Nosso fluxo na prática:**
* **GitHub:** Usamos como repositório remoto para hospedar e facilitar o compartilhamento do projeto.
* **Commits Semânticos:** Focamos em escrever mensagens claras nos *commits* para saber exatamente o histórico de evolução do robô.
* **Pull Requests e Branches:** Utilizamos ramificações (como no *merge* da branch do Luciano para a `main` no dia 21 de maio) para testar *features* (como a mira preditiva e o radar) antes de jogar na versão final.

## 🚀 O que aprendemos de verdade?
Esse projeto foi um marco super importante para consolidar o conceito e a prática de versionamento de código. Mais do que treinar a lógica do jogo, nós aprendemos a trabalhar em equipe usando as ferramentas essenciais do mercado. 

Entender como manter o histórico organizado, colaborar no mesmo código e gerenciar o repositório no GitHub foi o maior aprendizado que vamos levar para os nossos próximos projetos na área tech.
